# picoCTF - Beginner-Friendly CTF

**picoCTF** is a free Capture The Flag competition designed specifically for **beginners** and students, created by Carnegie Mellon University.

## Main Features

- **100% Free**: No costs or limitations
- **Permanently available**: Practice whenever you want
- **300+ Challenges**: Variety of categories and difficulties
- **Education oriented**: Designed to learn from scratch
- **Annual competitions**: Official events with prizes

## Challenge Categories

### Web Exploitation
- SQL Injection
- XSS (Cross-Site Scripting)
- Command Injection
- Authentication bypasses
- Cookie manipulation
- Server-Side vulnerabilities

### Cryptography
- Caesar cipher, ROT13
- RSA encryption
- Hash cracking
- Encoding/Decoding (Base64, Hex, etc.)
- Substitution ciphers
- Modern cryptography

### Forensics
- File analysis (metadata, strings)
- Image steganography
- Packet capture analysis (pcap)
- Memory dumps
- Disk forensics
- Log analysis

### Reverse Engineering
- Binary analysis
- Decompilation
- Static and dynamic analysis
- Assembly language (x86, ARM)
- Patching binaries
- Debugging with GDB

### Binary Exploitation (Pwn)
- Buffer overflows
- Format string vulnerabilities
- Return-oriented programming (ROP)
- Heap exploitation
- Stack canaries bypass

### General Skills
- Linux commands
- Python scripting
- Bash scripting
- Networking basics
- File manipulation

## Difficulty Levels

Challenges range from **very easy** to **hard**:

1. **Beginner** (50-100 pts): Basic concepts
2. **Easy** (100-200 pts): Requires basic knowledge
3. **Medium** (200-300 pts): Intermediate techniques
4. **Hard** (300-500 pts): Complex challenges

## Competition Format

### picoCTF Competition (Annual)
- **Duration**: 2 weeks in March/April
- **Participants**: Students from around the world
- **Prizes**: For top performers
- **Categories**: Middle School, High School, College

### picoGym (Always-On)
- Practice platform available 24/7
- All challenges from previous years
- No time limit
- Practice at your own pace

## Advantages for Beginners

1. **Gentle learning curve**: Starts very basic
2. **Hints available**: Clues if you get stuck
3. **Community writeups**: Detailed solutions
4. **Educational focus**: More than competing, learning
5. **No pressure**: Practice without time stress

### Basic
- **Linux Terminal**: Essential commands

### Forensics
- **Wireshark**: Network traffic analysis

## Suggested Progression

### Week 1: General Skills & Web
- Familiarize with Linux
- Learn basic commands
- Solve General Skills challenges
- Start with easy Web Exploitation

### Week 2: Cryptography
- Learn classic ciphers
- Practice encoding/decoding
- Understand hashing
- Basic crypto challenges

### Week 3: Forensics
- File analysis
- Basic tools (strings, file)
- Simple steganography
- Basic PCAPs

### Week 4: Reverse Engineering
- Assembly concepts
- Using Ghidra
- Debugging with GDB
- Static binary analysis

## Success Tips

1. **Read full description**: There are always clues
2. **Use Google**: Search concepts you don't know
3. **Take notes**: Document learned techniques
4. **Ask for hints**: Don't get frustrated, use clues
5. **Read writeups**: After solving, read other solutions
6. **Join Discord**: Very active and helpful community
