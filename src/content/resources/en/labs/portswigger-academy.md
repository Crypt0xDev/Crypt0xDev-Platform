# PortSwigger Web Security Academy

**100% free** training platform created by PortSwigger (creators of Burp Suite) with practical labs on web vulnerabilities.

## Features

- **Completely free**: No costs or limitations
- **150+ labs**: Covering all OWASP Top 10 vulnerabilities
- **Progressive difficulty**: From beginner to expert
- **Burp Suite integration**: Labs designed to practice with Burp
- **Free certification**: Certificate upon completing topics

## Topics Covered

### Injection Vulnerabilities
- SQL Injection
- OS Command Injection
- XXE (XML External Entity)
- SSTI (Server-Side Template Injection)
- Path Traversal

### Client-Side Vulnerabilities
- XSS (Cross-Site Scripting)
- CSRF (Cross-Site Request Forgery)
- CORS (Cross-Origin Resource Sharing)
- Clickjacking
- DOM-based vulnerabilities

### Authentication and Authorization
- Authentication vulnerabilities
- Access Control flaws
- JWT attacks
- OAuth authentication

### Business Logic
- Business logic vulnerabilities
- Race conditions
- Web cache poisoning

### Advanced
- Insecure Deserialization
- Server-Side Request Forgery (SSRF)
- HTTP Request Smuggling
- WebSockets security

## Difficulty Levels

1. **Apprentice**: Basic concepts, ideal for beginners
2. **Practitioner**: Intermediate level, requires prior knowledge
3. **Expert**: Complex challenges requiring advanced techniques
