# IppSec - HackTheBox Video Walkthroughs

IppSec is one of the most respected channels in the cybersecurity community, specialized in **HackTheBox machine walkthroughs**.

## Why IppSec?

- **Detailed explanations**: Not just shows how to solve machines, but WHY each step
- **Professional methodology**: Follows a systematic pentesting approach
- **Real techniques**: Uses tools and techniques that apply in the real world
- **Consistent quality**: High-quality videos with clear audio
- **Educational pace**: Not too fast, explains each concept

## Channel Content

### HackTheBox Walkthroughs
- **250+ machines solved** (Linux and Windows)
- From Easy to Insane level
- Retired machines (old machines available to everyone)
- Active machines (when officially retired)

### Techniques Covered

#### Enumeration
- Nmap and reconnaissance techniques
- Directory and subdomain fuzzing
- Web service analysis

#### Exploitation
- SQL Injection
- XSS, CSRF, SSRF
- Remote Code Execution
- Buffer Overflows
- Deserialization attacks

#### Post-Exploitation
- Linux privilege escalation
- Windows privilege escalation
- Active Directory attacks
- Pivoting and lateral movement

## Learning Tips

1. **Pause and replicate**: Don't watch the entire video at once, pause and try yourself
2. **Take notes**: Document new commands and techniques
3. **Repeat machines**: Redo old machines to reinforce
4. **Search alternatives**: Try solving in different ways
5. **Use ippsec.rocks**: If stuck, search for specific technique
