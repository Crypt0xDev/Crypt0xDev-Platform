# IppSec - HackTheBox Video Walkthroughs

IppSec es uno de los canales más respetados de la comunidad de ciberseguridad, especializado en **walkthroughs de máquinas de HackTheBox**.

## ¿Por qué IppSec?

- **Explicaciones detalladas**: No solo muestra cómo resolver las máquinas, sino POR QUÉ cada paso
- **Metodología profesional**: Sigue un enfoque sistemático de pentesting
- **Técnicas reales**: Usa herramientas y técnicas que se aplican en el mundo real
- **Calidad consistente**: Videos de alta calidad con audio claro
- **Ritmo educativo**: No va demasiado rápido, explica cada concepto

## Contenido del Canal

### Walkthroughs de HackTheBox
- **250+ máquinas resueltas** (Linux y Windows)
- Desde nivel Easy hasta Insane
- Retired machines (máquinas antiguas disponibles para todos)
- Active machines (cuando se retiran oficialmente)

### Técnicas Cubiertas

#### Enumeración
- Nmap y técnicas de reconocimiento
- Fuzzing de directorios y subdominios
- Análisis de servicios web

#### Explotación
- SQL Injection
- XSS, CSRF, SSRF
- Remote Code Execution
- Buffer Overflows
- Deserialization attacks

#### Post-Explotación
- Linux privilege escalation
- Windows privilege escalation
- Active Directory attacks
- Pivoting y lateral movement

## Tips de Aprendizaje

1. **Pausa y replica**: No veas el video completo de una vez, pausa e intenta hacerlo tú mismo
2. **Toma notas**: Documenta comandos y técnicas nuevas
3. **Repite máquinas**: Vuelve a hacer máquinas antiguas para reforzar
4. **Busca alternativas**: Intenta resolver de formas diferentes
5. **Usa ippsec.rocks**: Si te atascas, busca la técnica específica
