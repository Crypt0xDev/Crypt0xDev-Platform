# picoCTF - Beginner-Friendly CTF

**picoCTF** es una competición de Capture The Flag gratuita diseñada específicamente para **principiantes** y estudiantes, creada por Carnegie Mellon University.

## Características Principales

- **100% Gratuito**: Sin costes ni limitaciones
- **Permanentemente disponible**: Practica cuando quieras
- **300+ Challenges**: Variedad de categorías y dificultades
- **Orientado a educación**: Diseñado para aprender desde cero
- **Competiciones anuales**: Eventos oficiales con premios

## Categorías de Challenges

### Web Exploitation
- SQL Injection
- XSS (Cross-Site Scripting)
- Command Injection
- Authentication bypasses
- Cookie manipulation
- Server-Side vulnerabilities

### Cryptography
- Caesar cipher, ROT13
- RSA encryption
- Hash cracking
- Encoding/Decoding (Base64, Hex, etc.)
- Substitution ciphers
- Modern cryptography

### Forensics
- File analysis (metadata, strings)
- Image steganography
- Packet capture analysis (pcap)
- Memory dumps
- Disk forensics
- Log analysis

### Reverse Engineering
- Binary analysis
- Decompilation
- Static and dynamic analysis
- Assembly language (x86, ARM)
- Patching binaries
- Debugging with GDB

### Binary Exploitation (Pwn)
- Buffer overflows
- Format string vulnerabilities
- Return-oriented programming (ROP)
- Heap exploitation
- Stack canaries bypass

### General Skills
- Linux commands
- Python scripting
- Bash scripting
- Networking basics
- File manipulation

## Niveles de Dificultad

Los challenges van desde **muy fácil** hasta **difícil**:

1. **Beginner** (50-100 pts): Conceptos básicos
2. **Easy** (100-200 pts): Requiere conocimientos básicos
3. **Medium** (200-300 pts): Técnicas intermedias
4. **Hard** (300-500 pts): Challenges complejos

## Formato de Competición

### picoCTF Competition (Anual)
- **Duración**: 2 semanas en marzo/abril
- **Participantes**: Estudiantes de todo el mundo
- **Premios**: Para top performers
- **Categorías**: Middle School, High School, College

### picoGym (Always-On)
- Plataforma práctica disponible 24/7
- Todos los challenges de años anteriores
- Sin límite de tiempo
- Practica a tu ritmo

## Herramientas Recomendadas

### Básicas
- **Terminal Linux**: Comandos esenciales
- **Python**: Scripting y automatización
- **CyberChef**: Encoding/decoding online
- **Burp Suite**: Web exploitation

### Forensics
- **Wireshark**: Análisis de tráfico de red
- **Binwalk**: Análisis de archivos binarios
- **Strings**: Extraer strings de archivos
- **Exiftool**: Metadata de imágenes

### Reverse Engineering
- **Ghidra**: Decompilador gratuito (NSA)
- **GDB**: Debugger para binarios
- **Radare2**: Framework de reverse engineering
- **ltrace/strace**: Tracing de llamadas

### Crypto
- **RsaCtfTool**: Herramientas para RSA
- **John the Ripper**: Password cracking
- **Hashcat**: GPU-accelerated cracking
- **CyberChef**: Operaciones criptográficas

## Progresión Sugerida

### Semana 1: General Skills & Web
- Familiarízate con Linux
- Aprende comandos básicos
- Resuelve challenges de General Skills
- Inicia con Web Exploitation fáciles

### Semana 2: Cryptography
- Aprende cifrados clásicos
- Practica encoding/decoding
- Entiende hashing
- Challenges crypto básicos

### Semana 3: Forensics
- Análisis de archivos
- Herramientas básicas (strings, file)
- Steganography simple
- PCAPs básicos

### Semana 4: Reverse Engineering
- Conceptos de assembly
- Uso de Ghidra
- Debugging con GDB
- Análisis estático de binarios
