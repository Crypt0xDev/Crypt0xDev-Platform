# PortSwigger Web Security Academy

Plataforma de formación **100% gratuita** creada por PortSwigger (creadores de Burp Suite) con laboratorios prácticos sobre vulnerabilidades web.

## Características

- **Totalmente gratuito**: Sin costes ni limitaciones
- **150+ laboratorios**: Cubriendo todas las vulnerabilidades OWASP Top 10
- **Dificultad progresiva**: Desde principiante hasta experto
- **Integración con Burp Suite**: Labs diseñados para practicar con Burp
- **Certificación gratuita**: Certificado al completar topics

## Temas Cubiertos

### Vulnerabilidades de Inyección
- SQL Injection
- OS Command Injection
- XXE (XML External Entity)
- SSTI (Server-Side Template Injection)
- Path Traversal

### Vulnerabilidades del Lado del Cliente
- XSS (Cross-Site Scripting)
- CSRF (Cross-Site Request Forgery)
- CORS (Cross-Origin Resource Sharing)
- Clickjacking
- DOM-based vulnerabilities

### Autenticación y Autorización
- Authentication vulnerabilities
- Access Control flaws
- JWT attacks
- OAuth authentication

### Lógica de Negocio
- Business logic vulnerabilities
- Race conditions
- Web cache poisoning

### Avanzadas
- Insecure Deserialization
- Server-Side Request Forgery (SSRF)
- HTTP Request Smuggling
- WebSockets security

## Niveles de Dificultad

1. **Apprentice**: Conceptos básicos, ideal para principiantes
2. **Practitioner**: Nivel intermedio, requiere conocimientos previos
3. **Expert**: Challenges complejos que requieren técnicas avanzadas

## Ventajas para Pentesting

- Práctica realista sin riesgos legales
- Laboratorios actualizados con últimas técnicas
- Explicaciones detalladas de cada vulnerabilidad
- Soluciones disponibles si te atascas
- Perfecto para preparar certificaciones (OSWE, BSCP)

## Certificación BSCP

PortSwigger también ofrece la certificación **Burp Suite Certified Practitioner (BSCP)**:
- Examen práctico hands-on
- 4 horas de duración
- Demuestra habilidades reales de pentesting web
- Reconocida en la industria
